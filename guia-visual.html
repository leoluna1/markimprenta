🧮 Cotizador Automático

Los clientes pueden calcular precios al instante. Para ajustar los precios:

// En script.js, busca esta sección: const basePrices = { 'flyers': 0.15, // $0.15 por unidad 'tarjetas': 0.25, // $0.25 por unidad 'packaging': 0.50, // $0.50 por unidad 'etiquetas': 0.10, // $0.10 por unidad 'diseno': 50, // $50 servicio completo 'digital': 0.20, 'pop': 0.30 };

El cotizador también considera:

  • ✓ Material (Estándar, Premium, Lujo)
  • ✓ Tamaño (Pequeño, Mediano, Grande)
  • ✓ Cantidad (descuentos automáticos por volumen)

🎨 Personalización de Colores

Cambia los colores para que coincidan con tu marca:

/* En styles.css, al inicio del archivo: */ :root { --primary-color: #e74c3c; /* Rojo principal */ --secondary-color: #c0392b; /* Rojo oscuro */ --accent-color: #f39c12; /* Amarillo/naranja */ }

Solo cambia estos valores hexadecimales por los colores de tu marca y todo el sitio se actualizará automáticamente.

✅ Checklist Pre-Lanzamiento

Antes de publicar, verifica que todo esté correcto:

  • ☐ Información de contacto actualizada (dirección, teléfonos, email)
  • ☐ WhatsApp configurado con número correcto
  • ☐ Al menos 10-15 productos en el catálogo
  • ☐ Precios del cotizador actualizados
  • ☐ Colores personalizados con tu marca
  • ☐ Probado en navegador móvil
  • ☐ Probado en Chrome, Firefox y Safari
  • ☐ Todas las imágenes cargan correctamente
  • ☐ Formularios funcionan (si tienes backend configurado)
  • ☐ No hay errores en la consola del navegador (F12)
